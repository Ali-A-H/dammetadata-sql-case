# Dammetadata SQL Case

Dette er et lite SQL-prosjekt laget som en demonstrasjon av databasedesign og metadataregistrering.

Prosjektet er inspirert av arbeidsoppgaver knyttet til strukturering og registrering av informasjon om dammer.

## Innhold
- `schema.sql` oppretter databasen og tabellene
- `sample_data.sql` legger inn eksempeldata
- `spørringer.sql` viser relevante SQL-spørringer
- `er-diagram.png` viser datamodellen

## Tabeller
- kommune
- dam
- kulturminne
- metadata_registrering

## Formål
Å vise enkel modellering av damdata, kulturminneinformasjon og metadataregistrering i MySQL.
